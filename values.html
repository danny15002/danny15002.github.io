Values Tier Ranking System

Tier 1 (Ranks 1-10)
Tier 2 (Ranks 11-20)
Tier 3 (Ranks 21-30)
Tier 4 (Ranks 31-40)
Tier 5 (Any quantity)

How to Use:

Tier Buttons (1-5): Move value to bottom of that tier (reorders automatically)

↑/↓: Move up or down one position within the list

TOP: Send to very top and make Tier 1

BOT: Send to very bottom and make Tier 5

Smart Tiers: Values are grouped by tier. Any quantity per tier, automatically ordered!
